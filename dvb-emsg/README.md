# dvb-emsg (deprecated)

This crate has been renamed to [`mp4-emsg`](https://crates.io/crates/mp4-emsg). The `dvb-emsg` crate is now a thin re-export shim — all types and functions are identical to `mp4-emsg 0.1.0`. Please update your dependency to `mp4-emsg` directly; this shim will not receive new features.
