## Table 13 — Position
_§5.2.2.1, PDF pp. 19-19_

| Position | Label | Description |
|---|---|---|
| 0x0 | default | broadcaster's default position |
| 0x1 | top | subtitles are placed at the top of the screen |
| 0x2 | bottom | subtitles are placed at the bottom of the screen |
| 0x3 | speaker | close to speaker mouth |
| 0x4 | dynamic | position of each subtitle may vary |
| 0x5 - 0xF | reserved for future use | |
