## Table 55 — EncodingVersion
_§9.4.2.2, PDF pp. 61-61_

| Value | Encoding Version |
|---|---|
| 0x00 to 0xEF | TVA reserved. |
| 0xF0 | DVB profile of TVA MPEG_7 profile (BiM) ISO/IEC 15938-1 [10] as defined in the present document. |
| 0xF1 to 0xF7 | DVB reserved. |
| 0xF8 to 0xFF | User defined. |
