# ETSI TR 101 290 v1.4.1 — Measurement guidelines for DVB systems (TS monitoring checks)

**Provenance.** The canonical PDF **is** vendored in this repository
(`specs/etsi_tr_101_290_v01.04.01_dvb_measurement.pdf`) and is the
authoritative source. The tables below are **hand-transcribed** (not
extractor-generated): TR 101 290's measurement-check tables are
condition/parameter tables, outside the bit-syntax-table format that the
geometry-based extractor in `tools/dvb-si-audit/` targets (the extractor
finds 0 syntax tables in this document). Transcribed 2026-06-09. Wording,
numbering and punctuation are reproduced verbatim from the PDF, including
the decimal-comma timing values ("0,5 s") and mixed buffer-name casing
("TBsys"/"Tbsys", "MBn"/"Mbn"). Where a single table cell stacks several
alternative conditions, the alternatives are separated with `<br>` line
breaks; lines that merely wrap within one condition are joined.

Note on table numbering: v1.4.1 numbers the three priority tables
**5.0a / 5.0b / 5.0c** (earlier editions of TR 101 290 numbered them
5.1 / 5.2 / 5.3).
