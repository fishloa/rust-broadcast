# Table 31: Platform_ID registration template

_Source: specs/etsi_ts_101_162_v01.09.01_dvb_identifiers.pdf (PDF pp.12-44). NB: this is a registration-PROCESS spec (per-identifier registration/allocation templates + domain-name registries); live value allocations are maintained at the DVB online registry, not fixed here._


|  Registration field | Required | Description  |
| --- | --- | --- |
|  Platform Name | required | Name of the IP/MAC Platform (e.g. "ACME MobileTV").  |
|  Platform Operator | required | Name of organization which operates IP/MAC Platform (e.g. "ACME Mobile Com, Inc.")  |
|  Platform Legal Contact | required | Name and e-mail of authorized legal signatory of "Platform Operator"  |
|  Platform Technical Contact | required | Name and e-mail of technical contact of "Platform Operator"  |
|  Platform Notes | optional | Notes on the application, e.g. last revised and what revisions were made  |
