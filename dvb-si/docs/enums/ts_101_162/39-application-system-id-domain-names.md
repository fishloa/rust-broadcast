# 8 Identifiers for Globally Executable MHP (GEM), Multimedia Home Platform (DVB-MHP), and other technologies

# 8.0 Scope

The Application Information Table (AIT) provides full information on the data broadcast, the required activation state of applications carried by it, etc. Although first used with MHP (ETSI TS 101 812 [i.5] and ETSI TS 102 727 [i.6]), the AIT is now used with other technologies including OCAP™ [i.20], MHEG-5 [i.25] and various specifications which use ETSI TS 102 809 [i.24], for example the Open IPTV Forum specifications [i.21], [i.22] and [i.23]. In many places in the present document, the identifiers retain MHP in the name to reflect their origins but this does not imply limitations on their use with other application types.
