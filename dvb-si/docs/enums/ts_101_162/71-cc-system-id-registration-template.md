# Table 71: CC_System_ID registration template

_Source: specs/etsi_ts_101_162_v01.09.01_dvb_identifiers.pdf (PDF pp.12-44). NB: this is a registration-PROCESS spec (per-identifier registration/allocation templates + domain-name registries); live value allocations are maintained at the DVB online registry, not fixed here._


|  Registration field | Required | Description  |
| --- | --- | --- |
|  CC System Name | required | Name for the CC system.  |
|  CC System Specifier | required | Name of the organization which is responsible for the CC system.  |
|  CC System Legal Contact | required | Name and e-mail of authorized legal signatory of "CC System Name".  |
|  CC System Technical Contact | required | Name and e-mail of technical contact of "CC System Name".  |
|  CC System Notes | optional | Notes on the application, e.g. last revised and what revisions were made.  |

# 13.2.4 CC_System_ID allocation template

The scheme and values given in table 72 shall be used for the allocation of CC_System_ID values.
