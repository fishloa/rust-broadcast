|  Registration field | Required | Description  |
| --- | --- | --- |
|  Protocol Specification Name | required | Name of a Protocol Specification (e.g. "ACMEcast 1.0")  |
|  Protocol Specifier | required | Name of the organization specifying the "Protocol Specification Name" mentioned above (e.g. "ACMEcast, Inc.")  |
|  Protocol Legal Contact | required | Name and e-mail of authorized legal signatory of "Protocol Specifier"  |
|  Protocol Technical Contact | required | Name and e-mail of technical contact of "Protocol Specifier"  |
