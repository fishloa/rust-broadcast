## Table 111 — link_type
_§10.4.3, PDF pp. 98-98_

| Value | Description |
|---|---|
| 0x0 | Link information is a URI string only |
| 0x1 | Link information is a binary locator only |
| 0x2 | Link information is both a binary locator and a URI string |
| 0x3 | Link information is through means of a descriptor |
| 0x4 to 0xF | DVB reserved |
