## Table 28 — Result type
_§7.3.2.3.2, PDF pp. 40-40_

| result_type | Meaning |
|---|---|
| '00' | Result is a list of CRIDs, from one or more CRID authorities. |
| '01' | Result is a list of DVB binary encoded locators. |
| '10' | Result is a list of locators, in mixed format. See table 30. |
| '11' | DVB Reserved. |
