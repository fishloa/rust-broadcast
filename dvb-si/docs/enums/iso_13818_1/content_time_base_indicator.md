## Table 2-85 — Content_time_base_indicator values
_Rec. ITU-T H.222.0 (06/2021) | ISO/IEC 13818-1:2021 §2.6.57, Table 2-85; PDF p.100._

| Value | Description |
|---|---|
| 0 | No content time base defined in this descriptor |
| 1 | Use of STC |
| 2 | Use of NPT |
| 3..7 | Reserved |
| 8..15 | Use of privately defined content time base |
