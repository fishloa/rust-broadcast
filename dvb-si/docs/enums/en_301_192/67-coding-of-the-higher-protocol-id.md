## Table 67 — Coding of the higher_protocol_id
_§12.2.2, PDF pp. 75-75_

| higher_protocol_id | higher protocol |
|---|---|
| 0x00 | reserved |
| 0x01 | dGNSS data, see RTCM Recommended Standards for Differential GNSS Service [16] |
| 0x02 | TPEG, see CEN ISO/TS 18234 [17] |
| 0x03 .. 0x0F | reserved |
