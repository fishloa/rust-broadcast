## Table 1 — T2-MI packet types

| T2-MI packet_type | Description |
|---|---|
| 00₁₆ | Baseband Frame |
| 01₁₆ | Auxiliary stream I/Q data |
| 02₁₆ | Arbitrary cell insertion |
| 10₁₆ | L1-current |
| 11₁₆ | L1-future |
| 12₁₆ | P2 bias balancing cells |
| 20₁₆ | DVB-T2 timestamp |
| 21₁₆ | Individual addressing |
| 30₁₆ | FEF part: Null |
| 31₁₆ | FEF part: I/Q data |
| 32₁₆ | FEF part: composite |
| 33₁₆ | FEF sub-part |
