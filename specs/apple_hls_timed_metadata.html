Timed Metadata for HTTP Live Streaming

PDF Companion File

    1.0 Introduction

    Important: This document is no longer being updated. For the latest information about Apple SDKs, visit the documentation website.

    HTTP Live Streaming supports inclusion of timed metadata in ID3 format, carried in the MPEG-2 transport stream.

    This document describes how ID3 (reference [3]) metadata is carried as timed metadata in MPEG-2 Transport Streams (reference [1]) as used by the HTTP Live Streaming protocol (reference [2]). See section 3.0, References, for the relevant specifications.
