# Table 37: Application_System_ID registration template

_Source: specs/etsi_ts_101_162_v01.09.01_dvb_identifiers.pdf (PDF pp.12-44). NB: this is a registration-PROCESS spec (per-identifier registration/allocation templates + domain-name registries); live value allocations are maintained at the DVB online registry, not fixed here._


|  Registration field | Required | Description  |
| --- | --- | --- |
|  Application System Specification Name | required | Name of an Application System Specification (e.g. "ACME Internet TV 1.0")  |
|  Application System Specifier | required | Name of the organization specifying the "Application System Specification Name" mentioned above (e.g. "ACMEcast, Inc.")  |
|  Application System Legal Contact | required | Name and e-mail of authorized legal signatory of "Application System Specifier"  |
|  Application System Technical Contact | required | Name and e-mail of technical contact of "Application System Specifier"  |
|  Application System Notes | optional | Notes on the application, e.g. last revised and what revisions were made  |
