|  Registration field | Required | Description  |
| --- | --- | --- |
|  Geographical Area Name | required | Name of the geographical area (e.g. "North America")  |
|  Geographical Area Legal Contact | required | Name and e-mail of authorized legal signatory of "Geographical Area Name"  |
|  Geographical Area Technical Contact | required | Name and e-mail of technical contact of "Geographical Area Name"  |
|  Geographical Area Notes | optional | Notes on the application, e.g. last revised and what revisions were made  |
