# Client API 2.0 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers-extended-cc:subagent-driven-development (if subagents available) or superpowers-extended-cc:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Typed, trait-driven TS-stream parsing — `SiDemux`/`T2miPump` feed-and-iterate pumps, `AnyTable`/`AnyDescriptor`/`AnyPayload` dispatch generated from trait impls, `DvbText`/`LangCode` decoded serde — released as 2.0.0 lockstep.

**Architecture:** Four additive layers per the spec (`docs/superpowers/specs/2026-06-05-client-api-design.md`): owning-`Bytes` events with lazy zero-copy views, version-gated demux hot path, dispatch generated by one declarative macro per crate (no hand-written match), open registry escape hatch for client private tags.

**Tech Stack:** Rust 1.75 (MSRV — no newer features!), `bytes` (already a dep), `erased_serde` (new, optional, `serde`-gated). Workspace invariants: `cargo fmt --all --check`, clippy `-D warnings`, `RUSTDOCFLAGS="-D warnings"`, `--locked` everywhere, round-trip tests for every wire type, spec-cited doc comments.

**Branch:** `client-api-2.0`. Commit per task. Tracking: issue #16.

**Verified baseline facts** (read before coding, but these are confirmed):
- `dvb_si::ts::SectionReassembler::feed(&[u8], pusi)` + `pop_section() -> Option<bytes::Bytes>` — sections already come out as `Bytes`.
- `dvb_si::ts::TsPacket::parse(&[u8]) -> Result<TsPacket>` with `TsHeader { pid, pusi, … }`.
- `dvb_t2mi::ts::PacketReassembler::feed/pop_packet/drain_packets -> bytes::Bytes`.
- `dvb_common::Parse<'a> { type Error; fn parse(&'a [u8]) -> Result<Self, Self::Error> }`.
- A trait named `Descriptor` already exists in `dvb_si::traits` and an MPEG-2-subset enum named `Descriptor` exists in `descriptors/mod.rs` — new names are `DescriptorDef` / `AnyDescriptor` to avoid collision. The old subset enum is **removed** in 2.0 (Task 3) — it is subsumed by `AnyDescriptor`.

/// Borrowed DVB-encoded text (EN 300 468 Annex A). Wraps the raw selector +
/// body bytes; decoding happens only on [`DvbText::decode`] / `Display` /
/// serde — never in the parse hot path.
#[derive(Clone, Copy, PartialEq, Eq, Hash)]
pub struct DvbText<'a>(&'a [u8]);

impl<'a> DvbText<'a> {
    /// Wrap raw Annex A bytes (charset selector included, if any).
    #[must_use]
    pub const fn new(raw: &'a [u8]) -> Self {
        Self(raw)
    }
    /// The raw wire bytes, selector included.
    #[must_use]
    pub const fn raw(&self) -> &'a [u8] {
        self.0
    }
    /// Decode per Annex A (Table A.3 selector + control codes). Borrows for
    /// pure-ASCII input, allocates otherwise.
    #[must_use]
    pub fn decode(&self) -> std::borrow::Cow<'a, str> {
        decode(self.0)
    }
}

impl std::ops::Deref for DvbText<'_> {
    type Target = [u8];
    fn deref(&self) -> &[u8] {
        self.0
    }
}

impl std::fmt::Display for DvbText<'_> {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        f.write_str(&self.decode())
    }
}

impl std::fmt::Debug for DvbText<'_> {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "DvbText({:?})", self.decode())
    }
}

impl<'a> From<&'a [u8]> for DvbText<'a> {
    fn from(raw: &'a [u8]) -> Self {
        Self(raw)
    }
}

#[cfg(feature = "serde")]
impl serde::Serialize for DvbText<'_> {
    fn serialize<S: serde::Serializer>(&self, s: S) -> Result<S::Ok, S::Error> {
        s.serialize_str(&self.decode())
    }
}

/// ISO 639-2 language code or ISO 3166 country code — 3 raw bytes.
#[derive(Clone, Copy, PartialEq, Eq, Hash)]
pub struct LangCode(pub [u8; 3]);

impl LangCode {
    /// The code as a string; lossy (U+FFFD) for non-ASCII garbage.
    #[must_use]
    pub fn as_str(&self) -> std::borrow::Cow<'_, str> {
        String::from_utf8_lossy(&self.0)
    }
}

impl std::ops::Deref for LangCode {
    type Target = [u8; 3];
    fn deref(&self) -> &[u8; 3] {
        &self.0
    }
}

impl std::fmt::Display for LangCode {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        f.write_str(&self.as_str())
    }
}

impl std::fmt::Debug for LangCode {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "LangCode({})", self.as_str())
    }
}

#[cfg(feature = "serde")]
impl serde::Serialize for LangCode {
    fn serialize<S: serde::Serializer>(&self, s: S) -> Result<S::Ok, S::Error> {
        s.serialize_str(&self.as_str())
    }
}

#[cfg(feature = "serde")]
impl<'de> serde::Deserialize<'de> for LangCode {
    fn deserialize<D: serde::Deserializer<'de>>(d: D) -> Result<Self, D::Error> {
        let s = <&str>::deserialize(d)?;
        let b = s.as_bytes();
        if b.len() != 3 {
            return Err(serde::de::Error::invalid_length(b.len(), &"a 3-byte code"));
        }
        Ok(LangCode([b[0], b[1], b[2]]))
    }
}

```rust
macro_rules! declare_descriptors {
    ($lt:lifetime; $( $variant:ident => $($path:ident)::+ $(<$plt:lifetime>)? ),+ $(,)?) => {
        /// Every crate-implemented descriptor. External serde tagging,
        /// camelCase — `{"shortEvent": {…}}`.
        #[derive(Debug)]
        #[cfg_attr(feature = "serde", derive(serde::Serialize))]
        #[cfg_attr(feature = "serde", serde(rename_all = "camelCase"))]
        #[non_exhaustive]
        pub enum AnyDescriptor<$lt> {
            $( $variant($($path)::+ $(<$plt>)?), )+
            /// Tag with no typed implementation; `body` is the payload sans header.
            Unknown { tag: u8, body: &$lt [u8] },
        }

        $(
            impl<$lt> From<$($path)::+ $(<$plt>)?> for AnyDescriptor<$lt> {
                fn from(d: $($path)::+ $(<$plt>)?) -> Self { Self::$variant(d) }
            }
        )+

        impl<$lt> AnyDescriptor<$lt> {
            /// Parse one full descriptor (header included) by its tag.
            /// `None` = no typed impl for this tag (caller makes `Unknown`).
            pub(crate) fn dispatch(tag: u8, full: &$lt [u8]) -> Option<crate::Result<Self>> {
                use dvb_common::Parse;
                match tag {
                    $(
                        <$($path)::+ as crate::traits::DescriptorDef>::TAG
                            if tag != crate::descriptors::logical_channel::TAG =>
                        {
                            Some(<$($path)::+>::parse(full).map(Self::$variant))
                        }
                    )+
                    _ => None,
                }
            }
        }
    };
}
```

> **Implementation note:** match arms need *constant patterns*, not `const` paths with guards as sketched — the working form is to pass the tag literal in the macro line (`ShortEvent = 0x4D => short_event::ShortEventDescriptor<'a>`) and emit `0x4D => …` arms, with a unit test asserting `lit == <T as DescriptorDef>::TAG` for every line so the literal can never drift from the trait. Use that form. Exclude `LogicalChannel` from the generated match by simply not giving it a tag line (variant only — add it via a second macro section `@no_dispatch LogicalChannel => logical_channel::LogicalChannelDescriptor`).

- [ ] **Step 3: `parse_loop` + iterator** (`any.rs`)

```rust
/// Lazily walk a raw descriptor loop. Never panics. Per-descriptor parse
/// errors yield `Err` and iteration continues (the loop length field bounds
/// each entry); a truncated final header/body yields one `Err` then stops.
#[must_use]
pub fn parse_loop(bytes: &[u8]) -> DescriptorIter<'_> {
    DescriptorIter { bytes, pos: 0, fused: false }
}

#[derive(Debug, Clone)]
pub struct DescriptorIter<'a> {
    bytes: &'a [u8],
    pos: usize,
    fused: bool,
}

impl<'a> Iterator for DescriptorIter<'a> {
    type Item = crate::Result<AnyDescriptor<'a>>;

    fn next(&mut self) -> Option<Self::Item> {
        if self.fused || self.pos >= self.bytes.len() {
            return None;
        }
        let rem = &self.bytes[self.pos..];
        if rem.len() < 2 {
            self.fused = true;
            return Some(Err(crate::Error::BufferTooShort {
                need: 2, have: rem.len(), what: "descriptor header in loop",
            }));
        }
        let tag = rem[0];
        let len = rem[1] as usize;
        let total = 2 + len;
        if rem.len() < total {
            self.fused = true;
            return Some(Err(crate::Error::BufferTooShort {
                need: total, have: rem.len(), what: "descriptor body in loop",
            }));
        }
        let full = &rem[..total];
        self.pos += total;
        Some(match AnyDescriptor::dispatch(tag, full) {
            Some(res) => res, // Ok(typed) or Err(parse error) — either way, continue
            None => Ok(AnyDescriptor::Unknown { tag, body: &full[2..] }),
        })
    }
}
```

```rust
/// Object-safe face of a runtime-registered descriptor. Registered types are
/// OWNED (`'static`) — the `dyn Any` downcast requires it.
#[cfg(not(feature = "serde"))]
pub trait DescriptorObject: std::fmt::Debug + std::any::Any + Send + Sync {
    fn as_any(&self) -> &dyn std::any::Any;
}
#[cfg(feature = "serde")]
pub trait DescriptorObject:
    std::fmt::Debug + std::any::Any + Send + Sync + erased_serde::Serialize
{
    fn as_any(&self) -> &dyn std::any::Any;
}
// blanket impl for any qualifying T (one per cfg arm, same body):
//   impl<T: std::fmt::Debug + std::any::Any + Send + Sync [+ serde::Serialize]> DescriptorObject for T { … }

type CustomParse =
    Box<dyn for<'a> Fn(&'a [u8]) -> crate::Result<Box<dyn DescriptorObject>> + Send + Sync>;

#[derive(Default)]
pub struct DescriptorRegistry {
    custom: std::collections::HashMap<u8, CustomParse>,
    logical_channel: bool,
}

impl DescriptorRegistry {
    #[must_use] pub fn new() -> Self { Self::default() }

    /// Register an owned custom descriptor type for its `TAG`.
    pub fn register<T>(&mut self) -> &mut Self
    where
        T: for<'a> crate::traits::DescriptorDef<'a> + DescriptorObject + 'static,
    {
        self.custom.insert(
            <T as crate::traits::DescriptorDef>::TAG,
            Box::new(|b| {
                use dvb_common::Parse;
                Ok(Box::new(T::parse(b)?) as Box<dyn DescriptorObject>)
            }),
        );
        self
    }

        self.logical_channel = true;
        self
    }

    #[must_use]
    pub fn parse_loop<'r, 'a>(&'r self, bytes: &'a [u8]) -> RegistryIter<'r, 'a> { … }
}
```

- [ ] **Step 2: Demux core** — `SiDemux { pids: HashMap<Pid, SectionReassembler>, gate: HashMap<u64, GateEntry>, gate_order: VecDeque<u64>, cfg, stats, scratch: Vec<SectionEvent> }`.

`feed`:
```rust
pub fn feed(&mut self, packet: &[u8]) -> impl Iterator<Item = SectionEvent> + '_ {
    self.scratch.clear();
    self.stats.packets += 1;
    match crate::ts::TsPacket::parse(packet) {
        Err(_) => self.stats.malformed_packets += 1,
        Ok(ts) => {
            if let Some(reasm) = self.pids.get_mut(&ts.header.pid) {
                reasm.feed(ts.payload, ts.header.pusi);
                while let Some(section) = reasm.pop_section() {
                    self.stats.sections_completed += 1;
                    self.consider(ts.header.pid, section); // gate + CRC + push scratch
                }
            }
        }
    }
    // PAT-follow happens on the emitted PAT inside consider().
    self.scratch.drain(..)
}
```

`consider`: read header bits straight from the bytes (`section_syntax_indicator` = bit 7 of byte 1; long-form: table_id_extension bytes 3–4, version `(byte5 >> 1) & 0x1F`, section_number byte 6; CRC = last 4 bytes, validated with `dvb_common::crc32_mpeg2` — match `Section::validate_crc` semantics, including the TOT SSI=0-with-CRC exception already handled by `Section`). Gate key: `pid.value() as u64 | (table_id as u64) << 13 | (ext as u64) << 21 | (section_number as u64) << 37`. Short-form: ext=0, section_number=0, store crc32-of-all-bytes as the change detector. On change (or `emit_repeats`): push `SectionEvent`; if `follow_pat && table_id == 0x00`, parse `Pat` from the bytes and `self.pids.entry(pmt_pid).or_insert_with(SectionReassembler::new)` for each program entry with `program_number != 0`.
