# v8.3.0 — 2026-07-03

Lockstep release of the six DVB core crates (`broadcast-common`, `dvb-si`,
`dvb-t2mi`, `dvb-bbframe`, `dvb-conformance`, `dvb-tools`). Additive minor.

## Highlights

- **dvb-si — `StreamType` classifier helpers (#522).** `StreamType::is_video()`
  / `is_audio()` / `is_subtitle()` / `is_data()` over the ISO/IEC 13818-1
  Table 2-34 stream_type set (22 video / 9 audio / 1 subtitle variants;
  `is_data` is the complement). `PesPrivateData` (0x06) classifies as data, with
  a documented note that AC-3/E-AC-3/subtitle/teletext are disambiguated by the
  component/registration descriptors (ETSI TS 101 154), not the stream_type byte
  alone. An exhaustive `0x00..=0xFF` partition test guards the classification.

The other five lockstep crates carry no functional changes this release; they
bump in lockstep with `dvb-si`.

## Compatibility

Additive — no breaking changes. MSRV unchanged (1.81). All inter-crate
dependency references (`version = "8"`) continue to resolve.
