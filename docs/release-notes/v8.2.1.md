# v8.2.1 — 2026-07-02

Lockstep patch. **Documentation fix** for `broadcast-common`: the 8.2.0 release
shipped the new `mux` container-mux traits (`Unpackage`/`Package`/`Encrypt`/
`Decrypt`) but its published docs did not describe them. 8.2.1 documents the
`mux` module in the crate-root `//!` and README, so docs.rs and crates.io reflect
the API.

No API or behavioural change. The other lockstep crates (`dvb-si`, `dvb-t2mi`,
`dvb-bbframe`, `dvb-conformance`, `dvb-tools`) bump to 8.2.1 for lockstep parity
only — no source change.
