# Release wave — 2026-07-03 — MSRV 1.86 + edition 2024, DRM/KLV

Full-workspace republish so crates.io matches `main`: **Rust edition 2024** and
**MSRV 1.86** across every active crate, plus new transmux DRM/KLV features.
The 7 deprecated re-export shims (`dvb-common`, `dvb-scte35`, `dvb-pes`,
`dvb-cc`, `dvb-emsg`, `dvb-smpte2038`, `dvb-ule`) stay frozen (not republished).
