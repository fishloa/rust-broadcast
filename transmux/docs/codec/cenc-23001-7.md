# Common Encryption boxes (ISO/IEC 23001-7) — #429

**Source situation:** 23001-7 is paid (no free clean full-text; W3C EME-MP4 defers to
it). `sinf`/`frma`/`schm`/`schi`, `saiz`/`saio`, `sbgp`/`sgpd` are in the **vendored**
14496-12 (`isobmff-sample-groups.md`, §8.12 Protection Scheme). `tenc`/`senc`/`pssh`
layouts below are pinned by the **byte-exact ffmpeg oracle** (`fixtures/mp4/cenc.mp4`,
see `FMP4-GAPS-ORACLE.md`) + the well-established 23001-7 structure.

## tenc — TrackEncryptionBox  (oracle body `00000000 00 00 01 08 <16-byte KID>`)
```
aligned(8) class TrackEncryptionBox extends FullBox('tenc', version, 0) {
    unsigned int(8) reserved = 0;
    if (version==0) unsigned int(8) reserved = 0;
    else { unsigned int(4) default_crypt_byte_block; unsigned int(4) default_skip_byte_block; }  // cbcs
    unsigned int(8)  default_isProtected;
    unsigned int(8)  default_Per_Sample_IV_Size;
    unsigned int(8)  default_KID[16];
    if (default_isProtected==1 && default_Per_Sample_IV_Size==0){
        unsigned int(8) default_constant_IV_size; unsigned int(8) default_constant_IV[default_constant_IV_size];
    }
}
```

## senc — SampleEncryptionBox  (oracle `00000002 <IVs+subsamples>`, flags bit1 = UseSubSampleEncryption)
```
aligned(8) class SampleEncryptionBox extends FullBox('senc', 0, flags) {
    unsigned int(32) sample_count;
    for (i=0;i<sample_count;i++){
        unsigned int(8) InitializationVector[Per_Sample_IV_Size];
        if (flags & 0x000002) {   // UseSubSampleEncryption
            unsigned int(16) subsample_count;
            for (j=0;j<subsample_count;j++){
                unsigned int(16) BytesOfClearData;
                unsigned int(32) BytesOfProtectedData;
            }
        }
    }
}
```
`saiz`/`saio` (14496-12 §8.7.8-9) carry the per-sample aux-info sizes + the offset to
the `senc` aux data. `senc` is paired with a `seig` (`sgpd` grouping_type) for
constant-per-group encryption.

## pssh — ProtectionSystemSpecificHeaderBox
```
aligned(8) class ProtectionSystemSpecificHeaderBox extends FullBox('pssh', version, 0) {
    unsigned int(8) SystemID[16];
    if (version>0){ unsigned int(32) KID_count; { unsigned int(8) KID[16]; } [KID_count]; }
    unsigned int(32) DataSize;
    unsigned int(8)  Data[DataSize];   // DRM-system-specific init data
}
```
(ffmpeg emits no `pssh` without a DRM system → cover with a spec-vector unit test.)
