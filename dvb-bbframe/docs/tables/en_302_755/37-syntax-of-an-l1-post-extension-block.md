## Table 37 — Syntax of an L1-post extension block
_§7.2.3.4.0, PDF p. 74_

| Field | Length (bits) | Description |
|---|---|---|
| L1_EXT_BLOCK_TYPE | 8 | Indicates the type of L1-post extension block. See table 38. |
| L1_EXT_DATA_LEN | 16 | Indicates the length of the L1_EXT_BLOCK_DATA field in bits. |
| L1_EXT_BLOCK_DATA | Variable | Contains data specific to the type of L1-post extension block. |
